============
Installation
============
Best way to install pyAB is through pip

.. code:: python

   pip install pyAB

To install from source, use the following Github link

.. code:: python

   git clone https://github.com/AdiVarma27/pyAB.git
   cd pyAB
   python setup.py install


Dependencies
############

pyAB has the following dependencies:

- numpy
- matplotlib
- seaborn
- scipy
- statsmodels
